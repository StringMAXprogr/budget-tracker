# Budget Tracker (CLI)

## Overview

A simple Python command-line application to track personal income and expenses. This project is built for learning purposes and demonstrates basic programming concepts such as functions, loops, lists, and dictionaries.

---

## Features

* Add income transactions
* Add expense transactions
* View current balance
* View all transactions

---

## Project Structure

```
main.py
```

---

## Installation

Clone the repository:

```bash
git clone https://github.com/StringMAXprogr/budget-tracker.git
```

Navigate into the project folder:

```bash
cd budget-tracker
```

Make sure Python is installed:

```bash
python --version
```

---

## Usage

Run the program:

```bash
python main.py
```

Then follow the terminal menu to:

* Add income
* Add expenses
* View balance
* View transactions

---

## Example Menu

```
1. Add Income
2. Add Expense
3. Show Balance
4. Show Transactions
5. Exit
```
